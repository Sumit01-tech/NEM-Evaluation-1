# Project Setup
1. Clone the Repository: https://github.com/Sumit01-tech/NEM-Evaluation-1.git
2. Initial Setup:
   npm init -y
3. Install express
   npm install express
4. Install Nodemon Globally
   npm install -g nodemon
5. Start the server using nodemon server.js (as per the js file)
